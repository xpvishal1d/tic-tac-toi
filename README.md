# Tic Tac Toi

Single-project Next.js full-stack app with Tailwind, PostgreSQL, Drizzle, and cookie-based auth for a tic-tac-toe game.

## Quick start

1. `docker compose up -d`
2. `npm install`
3. `npm run db:push`
4. `npm run db:seed`
5. `npm run dev`

Default admin login:

- Email: `admin@tictactoi.local`
- Password: `Admin123!`
